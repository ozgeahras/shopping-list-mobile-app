## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/shopping-list-mobile-app.git
```

Go to the project directory

```bash
  cd ..movie-creator-with-OpenAI
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
